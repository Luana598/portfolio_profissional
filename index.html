Projetos

Potter.Search

Potter.Search é um projeto que consome a API pública PotterDB, exibindo informações sobre livros e filmes do universo Harry Potter. Desenvolvido com HTML5, CSS3 e JavaScript, utiliza manipulação do DOM e consumo de API para gerar conteúdo dinâmico. O site apresenta listas de livros e filmes, e permite visualizar detalhes como capa, resumo, autor, diretores e links externos.

Whatsapp API

Whatsapp API é um projeto desenvolvido em Node.js com Express.js, que fornece uma API para acessar dados de conversas armazenados localmente. Ela disponibiliza rotas para listar conversas e filtrar mensagens por palavra-chave, facilitando o acesso e a análise das informações.
